# 截图目录

将以下截图放入此目录：

| 文件名 | 内容 | 截图要求 |
|--------|------|----------|
| `01_final_submission.png` | 平台最终提交记录 | 显示队伍名、题号、提交时间、分数 |
| `02_environment.png` | 运行环境 | nvidia-smi / lscpu / 任务管理器 |
| `03_training.png` | 训练/推理过程 | 命令行、epoch 日志、validation 分数 |
| `04_file_hash.png` | 最终提交文件 SHA256 | 在终端中执行 sha256sum 并截图 |
| `05_fallback.png` | Fallback 证据（选交） | API 失败切换 / 格式错误修复等 |
| `06_a2_evidence.png` | A2 题额外记录（选交） | API 调用 / 模拟器通过 / schema 校验 |

### 截图规范

- 必须保留系统时间或终端时间戳
- 可打码 API key、用户名、私有路径
- 不得删除影响审核的时间、分数、命令、输出文件名
